# Future-Location-prediction-system-using-dataiku
Future Location Prediction System built using Dataiku DSS to analyze historical GPS data and predict a user’s next location. It uses data preprocessing, feature engineering, and machine learning models to deliver accurate, real-time insights for smart mobility and analytics applications.

🚀 Features
📊 Data ingestion from GPS datasets
🔧 Data preprocessing using DSS visual recipes
🧠 Machine learning (AutoML + optional Python models)
📍 Predict next user location (coordinates/clusters)
📈 Interactive dashboards & insights
⚡ Real-time prediction via API deployment

🛠️ Tech Stack
Dataiku DSS
Python (optional)
Scikit-learn / TensorFlow
GPS datasets

⚙️ How to Run (Using Dataiku DSS)
Install and open Dataiku DSS

Open Flow
